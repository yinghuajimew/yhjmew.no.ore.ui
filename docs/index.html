去 oreUI 打包

项目简介

去 oreUI 打包 是一个用于 Minecraft 基岩版的资源包项目。

功能说明

该项目的主要功能是将 Minecraft 中的 OreUI 界面风格还原为传统的 JsonUI 界面风格。

背景

OreUI 是 Minecraft 基岩版引入的新一代 UI 框架，部分玩家可能更习惯于经典的 JsonUI 界面风格。去 oreUI 打包 项目正是为这部分用户设计，允许他们恢复到之前的界面体验。

下载与使用

玩家可以从以下链接下载打包好的资源包：

https://openlist.endyun.ltd

兼容性

适用于 Minecraft 基岩版，建议在使用前确认游戏版本与资源包版本匹配。
